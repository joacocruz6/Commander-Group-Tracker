# Commander Group Tracker

## Motivation
I play commander a lot, so this project is to create a simple tracker for my play group. You can have an account and track your playgroup as well, I will do the backend and try to do some of the frontend as well.

## API
Aside from the login and those things, there will be APIs for:

1. CRU of a playgroup
1. CRUD of a person in a playgroup.
1. CRUD of a deck of a person in a playgroup.
1. CRU of a result of a match of a given playgroup.


## Limitations
A person in a playgroup is unique to and attached only to that playgroup, for now...
